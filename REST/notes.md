# Concepts

* Correct use of URIs
	- No verbs
	- Uniques
	- Format independent
	- Logical hierarchy
* Correct use of HTTP
	- Use of HTTP methods (POST, GET, PUT, DELETE, PATCH, HEAD)
	- Use of the HTTP error codes
* Hypermedia implementation
	- Comunication through xml (not very clear about how to use this though)
